# Локальная игра

Это текущая запускаемая локальная версия для проверки игровой логики.

## Запуск

1. Скачайте репозиторий через **Code → Download ZIP**.
2. Распакуйте архив.
3. Откройте файл `game/index.html` двойным кликом.

Никакие `npm install`, Node.js и BAT-файлы для этой версии не нужны.

## Управление

- Игрок 1: `WASD`, выстрел — `Space`.
- Игрок 2: стрелки, выстрел — `Enter`.
- `R` — перезапуск.
- `P` — пауза.

## Реализовано

- два игрока;
- выбор стартовой прокачки для каждого игрока;
- четыре типа врагов;
- движение врагов к базе;
- кирпич, сталь, вода, лес и лёд;
- база, жизни, бонусы, победа и поражение;
- бетонобойная максимальная пушка.

## Следующий этап

Эта версия используется как база логики. Дальше будет заменён визуальный слой: более классические спрайты танков, фиксированные карты и графика окружения ближе по ощущению к 8-битной игре.
