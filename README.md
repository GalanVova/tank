# Старые танки

Локальная и Telegram-версия кооперативной игры для двух игроков в духе классических 8-битных танков.

## Текущий статус

В репозиторий загружена рабочая базовая версия проекта v1.5:

- комнаты на двух игроков;
- серверная синхронизация через Socket.IO;
- выбор стартового уровня и прокачки обоих игроков;
- четыре типа вражеских танков;
- кирпич, сталь, вода, лес и лёд;
- база, жизни, бонусы, победа и поражение;
- локальный запуск на Windows/macOS;
- заготовка Telegram Mini App и Telegram-бота.

Это базовая версия для дальнейшей доработки. Следующий этап — переработка графики и фиксированных карт максимально близко по ощущению к классической игре, без копирования её оригинальных файлов и названия.

## Запуск

Требуется Node.js 20+.

```bash
npm install
npm start
```

Откройте:

```text
http://localhost:3000
```

Для проверки игры вдвоём откройте адрес в двух вкладках браузера.

## Управление

- Игрок 1: WASD + пробел.
- Игрок 2: стрелки + Enter.
- На телефоне используется экранное управление.

## Telegram

Создайте `.env` на основе `.env.example`:

```env
PORT=3000
BOT_TOKEN=token_from_botfather
WEBAPP_URL=https://your-public-https-domain
```

Запуск бота:

```bash
npm run bot
```
