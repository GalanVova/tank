# Старые танки

Локальная и будущая Telegram-версия кооперативной игры для двух игроков в духе классических 8-битных танков.

## Что уже лежит в репозитории

Текущая запускаемая версия находится здесь:

```text
game/index.html
```

Для запуска:

1. Нажмите **Code → Download ZIP**.
2. Распакуйте архив.
3. Откройте `game/index.html` двойным кликом.

Node.js и установка зависимостей для этой локальной версии не нужны.

## Управление

- Игрок 1: `WASD`, выстрел — `Space`.
- Игрок 2: стрелки, выстрел — `Enter`.
- `R` — перезапуск.
- `P` — пауза.

## Что реализовано

- два игрока на одном экране;
- отдельная стартовая прокачка каждого игрока;
- обычный, быстрый, усиленный и тяжёлый враги;
- движение врагов к базе и стрельба;
- кирпич, сталь, вода, лес и лёд;
- база, жизни, бонусы, победа и поражение;
- максимальная бетонобойная пушка.
